# KI Website Auditor

Ein modernes Tool für SEO- und Performance-Audits, entwickelt mit Next.js und der Google PageSpeed ​​API.

## Features
- SPerformance scoring
- SEO analysis
- Zugänglichkeit
- Bewährte Verfahren(Modern responsive UI)

## Tech Stack
- Next.js
- TailwindCSS
- Google PageSpeed API
- Vercel
